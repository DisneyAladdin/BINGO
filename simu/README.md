BINGO
====
This is the BINGO game.
You can use easily once install this.
This game is for iOS phone. 
シンプルなビンゴゲームです．
既に出た数字の検索もできます．
iOS向けのゲームです.

# Usage
Just push the Button.
If you search the number, just enter the number and push "Search" Button.
